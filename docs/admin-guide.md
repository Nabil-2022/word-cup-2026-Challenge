# Administrator Guide

## Back Office

1. Log in at `/admin/login`.
2. Review KPIs on the dashboard.
3. Manage participants, matches, entries, payments, results, and compliance logs.
4. Create or import matches before opening the Prediction Challenge.
5. Enter official match scores.
6. Run the Scoring Engine.
7. Generate the Leaderboard.
8. Export CSV files when needed.

Sensitive admin actions are recorded in audit logs.

No winner is selected by random chance. The final ranking is based on score, tie-break distance, and validation time.
